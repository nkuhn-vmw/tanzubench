# Project Helios: Event Timeline

1. **2019-03-04** — Helios project charter signed by executive sponsor Dr. Yuen. Initial budget: $4.2M.
2. **2019-07-22** — First engineering prototype demonstrated to steering committee. Proof-of-concept declared successful.
3. **2020-01-15** — Team size doubled from 8 to 16 engineers following Series A close.
4. **2020-09-30** — Integration partner Axiom Networks withdrew from the program, citing conflicting product roadmaps.
5. **2021-02-11** — Helios v0.9 beta released to 50 design partners for pilot testing.
6. **2021-08-03** — Critical security vulnerability discovered in the authentication module; beta program paused for 6 weeks.
7. **2021-09-17** — Security patch certified by third-party auditor; beta program resumed.
8. **2022-04-28** — Helios v1.0 released to general availability. First commercial revenue recorded.
9. **2023-06-01** — Project transferred to new business unit (Enterprise Products) following internal reorganization.
10. **2024-11-19** — Helios v2.0 announced with AI-assisted anomaly detection; launch scheduled for Q2 2025.
